# MCPM-Lib    
BlockLauncherScriptModでVector3やPlayerをPocketMine風に扱えるようにする物です。
MITライセンスなのでModに組み込んで使用できます。

PocketMine-MPのドキュメントを参考にしてください。

## MCPM 0.1 〜

  - getY  
  - getZ  
  - multiply
  - divide
  - ceil
  - floor
  - round
  - abs
  - getRight
  - getUp
  - getForward

  - getWest
